# 🍳 Recipe Finder

A React web app that lets you search for recipes using the TheMealDB API. Get instant results with recipe images and names.

## Features

- Search recipes by name or ingredient
- Displays recipe cards with images
- Loading state while fetching results
- "No recipes found" message for invalid searches
- Responsive grid layout

## Tech Stack

- React
- Bootstrap 5
- TheMealDB API

## Getting Started

Clone the repository:
git clone https://github.com/Yashika-Nehra/recipe-finder.git

Install dependencies:
npm install

Run the app:
npm start

Open http://localhost:3000 in your browser.

## API

This project uses the free [TheMealDB API](https://www.themealdb.com/api.php) — no authentication required.
